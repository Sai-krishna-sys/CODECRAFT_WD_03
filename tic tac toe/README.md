# Tic-Tac-Toe Web Application

A simple tic-tac-toe game built with HTML, CSS, and JavaScript.

## Features

- Player vs Player mode
- Player vs AI mode (AI makes random moves)
- Interactive 3x3 grid
- Win detection and draw detection
- Reset functionality

## How to Play

1. Open `index.html` in a web browser.
2. Choose a game mode: Player vs Player or Player vs AI.
3. Click on the cells to place your marker (X or O).
4. The first player to get three in a row (horizontally, vertically, or diagonally) wins.
5. If all cells are filled without a winner, it's a draw.
6. Click "Reset Game" to start a new game.

## Files

- `index.html`: The main HTML structure
- `styles.css`: CSS styling for the game
- `script.js`: JavaScript logic for game functionality

## Running the Application

Simply open `index.html` in any modern web browser. No server required.
